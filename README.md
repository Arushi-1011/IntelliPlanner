# IntelliPlanner

A productivity planner built from scratch with vanilla HTML, CSS and JavaScript.

🔗 **Live demo:** https://arushi-1011.github.io/IntelliPlanner/

## Features

- *AI Task Breakdown* — describe any goal and get it broken into prioritized, estimated subtasks
- *Smart Priority Scoring* — every task gets a 0–99 score based on urgency and priority level
- *Pomodoro Focus Timer* — 25/5/15 mode switching with animated ring, session tracking and task selection
- *Habit Tracker* — daily check-ins, streak tracking and consistency visualization
- *Analytics Dashboard* — weekly bar chart, activity heatmap, priority and category breakdowns
- *Task Manager* — add, filter, search and delete tasks with live stats

## Tech

- Vanilla HTML, CSS, JavaScript — no frameworks, no dependencies
- CSS custom properties for theming
- Google Fonts (Syne + DM Sans)
- Deployed on GitHub Pages

## Run locally

Just open `index.html` in your browser. No build step, no install needed.
